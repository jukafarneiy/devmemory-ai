# Marketplace media

Drop captured screenshots and demo GIFs here, using the filenames defined in [`docs/marketplace-assets.md`](../../../../docs/marketplace-assets.md).

This folder is **excluded from the VSIX** (`.vscodeignore` rule `media/marketplace/**`) so listing assets do not bloat the extension package. Marketplace listings render images by absolute URL — once captured, reference them via the GitHub raw URL of this folder, not via relative paths.

Do **not** delete this folder or this file: the structure is what the asset checklist refers to.
