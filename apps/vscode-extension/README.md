# DevMemory AI VS Code Extension

Open the **DevMemory AI** view in the Activity Bar — the sidebar guides you through every step. The most common actions:

- **Set Up Memory** — scan the workspace and create `.ai-memory/`.
- **Teach DevMemory About This Project** / **Save Project Understanding** — bootstrap the memory with the help of an AI assistant.
- **Start AI Session** — copy a fresh resume prompt before working with your AI.
- **End AI Session** / **Save Session Summary** — wrap up a session and persist the AI's structured response.
- **Check Memory Quality** — audit the local memory store.

The extension stores memory locally in `.ai-memory/` inside the active workspace. No data leaves your machine.
